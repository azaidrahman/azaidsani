Setting Up This Blog with Cloud Run

January 24, 2026

I wanted a dead-simple blog where I could write weekly reflections. No frameworks, no build steps, just HTML files served by nginx.

The Stack

It's about as minimal as it gets:

  • Plain HTML + CSS
  • Nginx in a Docker container
  • Cloud Run for hosting
  • Cloud Build for CI/CD

How It Works

I write a new .html file, push to main, and Cloud Build automatically builds the container and deploys to Cloud Run. The whole deploy takes about 90 seconds.

Cost

With Cloud Run's generous free tier and min-instances set to 0, this should cost almost nothing. Maybe a few cents a month unless I get unexpectedly popular.
